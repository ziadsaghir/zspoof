# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   |      check         |
| < 1.0   |        x           |

## Reporting a Vulnerability

If you discover a security vulnerability within ZSPOOF (e.g., privilege escalation, buffer overflow in the C++ engine), please **DO NOT** open a public issue.

Instead, please email the lead developer at:
**[ziadsghir8@gmail.com]**

Please include:
* Description of the vulnerability.
* Steps to reproduce.
* Potential impact.

We will review the issue and publish a fix as soon as possible.
